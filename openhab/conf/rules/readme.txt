Your rules go here.
All rule files have to have the ".rules" file extension and must follow a special syntax.

Check out the openHAB documentation for more details:
https://www.openhab.org/docs/configuration/rules-dsl.html


Winter settings:
Disable dynamic scripts by changing file names "noon_dynamic" and "sunrise_dynamic" to .rule

Summer settings:
Enable dynamic scripts by changing file names "noon_dynamic" and "sunrise_dynamic" to .rules
